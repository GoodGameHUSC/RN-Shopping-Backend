# Shopping Me Backend 
 
RESTful API for backend project.

## Installation

Install the following tools in your system:

* Node - v8.9.4
* NPM - v6.2.0
* MongoDB - v4.0.1
* `npm install`
* `npm run dev`

## Tests

Tests have been written with mocha framework. For running test please use: `npm run test`.
